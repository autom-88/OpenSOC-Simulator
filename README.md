# OpenSOC Simulator

Educational SOC simulator scaffold.

🛡 OpenSOC Simulator

> An interactive Security Operations Center (SOC) simulator built with HTML, CSS and JavaScript.

OpenSOC Simulator is an educational web application designed to simulate the workflow of a modern Security Operations Center (SOC). The project provides an interactive environment where students, developers and cybersecurity enthusiasts can explore security monitoring, incident response and threat hunting concepts without requiring a real SIEM infrastructure.

---

🚀 Features

- Interactive SOC Dashboard
- Incident Management Center
- Threat Hunting Console
- Analytics Rules
- Entities Explorer
- Devices Inventory
- User Monitoring
- Security Timeline
- Playbooks (SOAR Simulation)
- Reports Center
- Settings Panel
- Live Event Simulation
- KQL Query Emulator
- Responsive Interface
- Dark Theme

📊 Modules

### Dashboard
Real-time overview of the simulated SOC environment.

### Incidents
Monitor and investigate simulated security incidents.

### Threat Hunting
Execute KQL-inspired queries against simulated security data.

### Analytics
Manage detection rules and alert generation.

### Entities
Search and inspect users, hosts, IP addresses and processes.

### Devices
Monitor endpoint inventory and security status.

### Users
Review user accounts, roles and authentication activity.

### Timeline
Visualize events in chronological order.

### Playbooks
Simulate automated SOAR response workflows.

### Reports
Generate and review incident reports.

### Settings
Customize simulator preferences and behavior.
